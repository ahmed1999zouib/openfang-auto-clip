# OpenFang Auto Clip v0.2.0 发布说明

发布日期：2026-02-28

## 本次版本重点

v0.2.0 引入了“版权安全转换”这一核心定位，用来帮助创作者和运营团队把长视频转成更适合平台传播的短视频资产。

## 主要能力

- Level 1 视觉转换框架
- 多平台短视频导出
- 自动化视频下载
- FFmpeg 处理管线
- 中英文项目文档

## 对外表达建议

- 强调 local-first 与 reproducible benchmark
- 不要把 Level 2 / Level 3 说成已完成商业能力
- 对外应把“版权风险控制”表述为工作流与策略，而不是法律承诺

## 推荐受众

- 内容创作者
- 教育内容团队
- 代运营与代理商
- 想研究本地化视频自动化工作流的开发者
