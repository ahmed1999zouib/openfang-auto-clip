# 🌐 Web管理界面使用说明

## 🚀 快速启动

### 方法1：一键启动（推荐）
```bash
./start_web_manager.sh
```

### 方法2：手动启动
```bash
source venv/bin/activate
python3 web_manager.py
```

启动后浏览器会自动打开：http://localhost:5000

---

## 🎨 界面功能

### 1️⃣ IP角色生成
- **🎨 生成IP角色** - 一键生成原创卡通角色
- **🌐 打开测试页面** - 在浏览器中测试AI生成

### 2️⃣ 视频处理
- **🔄 版权转换** - 测试Level 1版权转换效果
- **✂️ 创建片段** - 从转换后视频创建短视频

### 3️⃣ Level 3 重制
- **📦 生成项目包** - 生成完整项目（脚本+角色+场景）

### 4️⃣ YouTube处理
- **📹 视频URL** - 输入YouTube链接
- **🎚️ 转换级别** - 选择Level 1/2/3
- **🚀 开始处理** - 自动下载、转换、剪辑

### 5️⃣ 实时日志
- 查看所有操作日志
- 任务状态实时更新
- 错误信息显示

---

## 💡 使用流程

### 创建你的IP角色

1. **点击 "🎨 生成IP角色"**
   - 等待几秒完成
   - 自动创建角色设计

2. **点击 "🌐 打开测试页面"**
   - 浏览器打开新标签
   - 显示AI提示词

3. **在测试页面中**
   - 点击"复制提示词"
   - 在Bing Image Creator中粘贴
   - 生成你的原创角色！

### 处理YouTube视频

1. **输入YouTube URL**
   - 粘贴视频链接

2. **选择转换级别**
   - Level 1: 视觉转换（快速）
   - Level 2: 脚本重写（安全）
   - Level 3: 完全重制（最安全）

3. **点击"🚀 开始处理"**
   - 自动下载视频
   - 应用版权转换
   - 生成短视频片段

## 📊 界面说明

### 统计卡片
- **运行任务** - 当前运行的任务数
- **IP角色** - 已生成的角色数量
- **系统状态** - 服务器运行状态

### 操作卡片
每个功能都有独立的卡片：
- 功能图标和标题
- 操作按钮
- 实时状态显示
- 进度提示

### 日志面板
- 实时显示所有操作
- 时间戳标记
- 颜色编码（成功/错误）

---

## ⚡ 特色功能

### ✅ 一键操作
所有功能都是点击按钮即可完成，无需命令行！

### 🔄 实时更新
- 任务状态每5秒自动刷新
- 无需手动刷新页面

### 📋 可视化日志
- 所有的操作都有日志记录
- 实时查看任务进度

### 🎯 智能提示
- 操作成功/失败有明确提示
- 错误信息清晰显示

---

## 🔧 高级功能

### 后台任务
所有任务都在后台运行：
- 点击按钮后立即返回
- 继续其他操作
- 实时查看进度

### 多任务并发
可以同时运行多个任务：
- 生成IP角色同时
- 处理视频同时
- 创建片段同时

### 任务历史
所有任务都有记录：
- 任务ID
- 开始/结束时间
- 输出日志
- 错误信息
- 持久化状态文件：`~/.openfang/web_tasks.json`

### 可迁移部署
- 自动识别当前仓库目录，不再依赖硬编码绝对路径
- 可通过 `OPENFANG_AUTO_CLIP_PROJECT_DIR` 覆盖项目目录
- 可通过 `OPENFANG_OUTPUT_DIR` 覆盖输出目录

## 🛑 停止服务器

在终端中按：
```
Ctrl + C
```

服务器会停止，但已生成的文件会保留。

## 🐛 故障排除

### 端口被占用
如果5000端口被占用，修改 `web_manager.py`:
```python
app.run(host='127.0.0.1', port=5001, debug=False)
```

### Flask未安装
```bash
source venv/bin/activate
pip install -r requirements.txt
```

### 虚拟环境丢失
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
