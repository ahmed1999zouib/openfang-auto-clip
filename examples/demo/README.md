# Demo Walkthrough

This folder is a lightweight evaluation guide for first-time visitors.

## What this demonstrates

- the real CLI entrypoint
- the directory structure generated by a successful run
- the report shape a downstream operator or publisher can consume

## Suggested command

```bash
./auto_clip.sh "https://www.youtube.com/watch?v=VIDEO_ID" --transform 1 --duration 45
```

## Expected output structure

```text
~/.openfang/clips/
├── downloads/
│   └── source_video.mp4
└── 20260307_120000/
    ├── clip_01_0000s-0045s.mp4
    ├── clip_02_0045s-0090s.mp4
    ├── clip_03_0090s-0135s.mp4
    └── report.json
```

## Sample report

See [`sample_report.json`](sample_report.json) for an illustrative payload.

The sample is synthetic. It is included to make the project easier to evaluate without downloading media during a quick repo review.
