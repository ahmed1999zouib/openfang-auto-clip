# IP Protection and Character Copyright Guide

## What this document is for

This guide explains a hard constraint in the project: `Level 1` transformation can change style, pacing, and presentation, but it does not make a recognizable copyrighted character safe for commercial reuse.

## Core takeaway

- `Level 1` is a visual remix layer, not a legal shield
- recognisable cartoon characters, mascots, and branded visual identities still carry IP risk
- commercial use should prefer `Level 2` or `Level 3`

## Risk by transformation level

| Level | What changes | What stays risky |
|------|--------------|------------------|
| 1 | color, speed, mirror, crop, grading | original character identity can remain obvious |
| 2 | script, narration, assets, delivery | requires discipline in asset replacement and sourcing |
| 3 | full recreation with new characters and scenes | lowest IP risk when executed with original assets |

## Example

If a source video uses a well-known children's character, flipping the frame and changing the color curve still leaves you with the same recognizable character. That can still trigger trademark, character copyright, or brand-confusion problems.

## Recommended operating policy

### For hobby testing

- use `Level 1` only on low-risk, factual, non-character-heavy media
- do not market the result as fully original

### For commercial publishing

- use `Level 2` when you can rewrite the script and replace the media with licensed assets
- use `Level 3` when you need a fully original character system and safer long-term distribution

## Safe production pattern

1. extract the educational or informational intent from the source
2. rewrite the explanation in a new script
3. replace visuals with licensed or original assets
4. generate fresh narration and captions
5. publish the result as a derivative editorial workflow, not a disguised copy

## Suggested tool stack for safer output

- script generation: LLM workflow
- voiceover: ElevenLabs or OpenAI TTS
- image generation: DALL-E, Midjourney, or Stable Diffusion
- animation: Runway, Pika, or manual editing
- music: licensed library or original composition

## Bottom line

If the source depends on a protected character, mascot, or franchise identity, `Level 1` is not enough. Treat `Level 2` as the minimum commercial path and `Level 3` as the strong open-source reference architecture for truly original output.
