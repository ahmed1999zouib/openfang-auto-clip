🚀 Quick Start

1

Clone Repository

git clone https://github.com/YOUR_USERNAME/openfang-auto-clip.git
2

Install Dependencies

cd openfang-auto-clip && ./scripts/install.sh
3

Start Editing

./auto_clip.sh "https://youtube.com/watch?v=VIDEO_ID" --transform 1

Requirements

  • ✅ macOS or Linux
  • ✅ Python 3.9+
  • ✅ OpenFang 0.1.9+
  • ✅ FFmpeg (auto-installed)
