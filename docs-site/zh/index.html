先讲清楚已交付能力，再谈路线图

今天就能用的部分

  • 下载、切片并导出竖屏短视频
  • 基于 FFmpeg 的 Level 1 视觉转换
  • 带持久化任务记录的本地 Web 管理器
  • benchmark、storyboard、release、launch kit 工具链

仍在推进中的部分

  • Level 2 脚本重写仍是脚手架
  • Level 3 完全重制仍属于 roadmap
  • 当前不提供托管云 API 或 SaaS
  • 商业级信任还需要更强的 demo 和案例

快速评估流程

三条命令就能判断项目值不值得试

1. 环境检查

./auto_clip.sh --doctor

确认 Python、ffmpeg、yt-dlp、配置路径和输出目录是否就绪。

2. 跑 Benchmark

python3 scripts/run_demo_benchmark.py

生成 clips、`benchmark_report.json`、`preview.png` 和 `storyboard.png`，全程不依赖外部素材。

3. 生成发布文案

python3 scripts/generate_launch_kit.py --report examples/benchmark/sample_benchmark_report.json

根据 benchmark report 生成 launch 文案，让对外传播和仓库真实产物保持一致。
