## 🆘 Common Issues & Solutions

### Issue: Can't push to GitHub
```bash
# Check remote
git remote -v

# Fix URL
git remote set-url origin https://github.com/YOUR_USERNAME/openfang-auto-clip.git
```

### Issue: Large files error
```bash
# .gitignore should handle .mp4 files
# If stuck:
git rm --cached *.mp4
git commit -m "Remove video files"
```

### Issue: Branch name
```bash
# Rename to main
git branch -M main
git push -u origin main
```
