# 项目摘要

## 项目概况

- 名称：OpenFang Auto Clip
- 当前状态：持续迭代中
- 定位：本地优先的视频二次分发与短视频切条流水线
- 许可证：MIT

## 核心价值

- 把长视频拆成适合平台传播的短视频片段
- 通过 Level 1 视觉转换降低内容直接搬运的重复感
- 用 benchmark、storyboard、launch kit 和 release flow 增强仓库的可验证性

## 当前已交付

- CLI 主流程
- Level 1 视觉转换
- Web 管理器与任务状态持久化
- 合成 benchmark demo
- storyboard 图片输出
- launch kit 生成
- tag release 自动化

## 当前边界

- Level 2 / Level 3 仍属于脚手架或 roadmap
- 不提供托管云端服务
- 商业可信度仍需靠案例和 demo 继续加强

## 近期优先级

- 文档双语化
- 对外展示层持续优化
- benchmark 资产和传播素材增强
- 发布链路标准化
