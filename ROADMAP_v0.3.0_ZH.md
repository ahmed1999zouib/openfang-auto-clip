# v0.3.0 路线图

## 目标主题

把项目从“能运行的原型”继续推进到“更像一个可靠的开源工作流产品”。

## 核心方向

### 1. 交付可靠性

- 更稳的本地执行层
- 更清晰的配置与状态文件
- 更完整的测试覆盖

### 2. 传播与展示

- 更强的 benchmark 和 demo 资产
- 更可信的 landing page
- 更顺滑的发布链路

### 3. 工作流深化

- Level 1 继续优化
- Web 管理器继续增强
- 为 Level 2 / 3 做更清晰的实现边界和路线拆分

## 判断标准

- 新访客能在数分钟内跑通 benchmark
- tag release 流程可重复执行
- 中英文文档入口齐全
- README、landing page 与实际能力保持一致
